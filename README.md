# 🌊 NewsWave - AI-Powered News Intelligence Hub

> **AI-powered global news platform with real-time aggregation across 50+ sources, NLP sentiment analysis, and Gemini-powered article summarization.**

[![React](https://img.shields.io/badge/React-18.2-blue)](https://reactjs.org/)
[![Gemini AI](https://img.shields.io/badge/Gemini-AI--Powered-purple)](https://ai.google.dev/)
[![Firebase](https://img.shields.io/badge/Firebase-Backend-orange)](https://firebase.google.com/)

## ✨ Features

### 🤖 **AI-Powered Intelligence**
- **Gemini AI Summarization** - Get 3-bullet summaries of any article
- **NLP Sentiment Analysis** - Automatic positive/neutral/negative classification
- **Bias Detection** - Identifies editorial bias levels in articles

### 🌍 **Multi-Language Support**
- Support for 10+ languages (English, Hindi, Spanish, French, Arabic, Chinese, etc.)
- Real-time language switching
- Localized UI and content

### 🎯 **Advanced Features**
- Voice search with speech recognition
- Advanced filtering (date, sentiment, source)
- Real-time news from multiple sources
- YouTube video integration
- Offline PWA support
- User personalization with Firebase

### 🎨 **Premium UI/UX**
- Glassmorphism design
- Animated background gradients
- Skeleton loading states
- Dark/Light mode
- Fully responsive

## 🛠️ Tech Stack

**Frontend:**
- React 18 + Vite
- Tailwind CSS
- i18next (multi-language)
- Framer Motion (animations)

**AI/ML:**
- Google Gemini AI
- NLP sentiment analysis
- Bias detection algorithms

**Backend & Services:**
- Firebase (Firestore, Auth)
- The Guardian API
- YouTube Data API v3

## 🚀 Quick Start
```bash
# Install dependencies
npm install

# Setup environment variables (see .env.example)
cp .env.example .env

# Run development server
npm run dev

# Build for production
npm run build
```

## 🔑 API Keys Required

1. **Gemini AI**: https://makersuite.google.com/app/apikey
2. **YouTube API**: https://console.cloud.google.com/
